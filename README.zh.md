## 项目描述

胡志明共产主义青年团 - 越南学生会管理系统是一个数字化转型解决方案,旨在规范青年团和学生会组织的数据管理流程和业务工作流程。该软件支持学生档案管理、班级干部人员跟踪和内部活动协调。

**重要提示:** 这是为胡志明共产主义青年团(HCYU)和越南全国学生联合会(VNUS)组织开发的内部软件。组织外使用需要审查和批准。

## 主要功能

### 1. 学生和团员档案管理
- 学生和团员信息完全数字化
- 添加、更新或删除数据条目
- 按学号、姓名或班级快速搜索和查询
- 以Excel格式导入数据和导出报告
- 特殊备注和实体档案存储位置管理

### 2. 班级单位和青年干部管理
- 管理分支协会和班级列表,按院系和学年分类
- 跟踪分支协会中的执行委员会和运营委员会系统
- 监控工作任期,支持竞赛活动和表彰计划
- 青年干部规划和培养

### 3. 办公室和内部行政任务
- 内部通知系统和行政文档共享
- 任务分配(工作委派)和进度跟踪
- 运营工作流程管理

### 4. 数据存储和安全
- 数字文档存储库和附件文件管理
- 备份和数据恢复机制
- 变更历史记录用于审计和监控

## 系统要求

### 最低配置:
- **操作系统:** Windows 10 (64位) 或更高版本
- **内存:** 4 GB
- **存储空间:** 500 MB 可用空间
- **显示器:** 1366x768 分辨率
- **互联网:** 稳定连接(实时数据同步所需)

### 推荐配置:
- **操作系统:** Windows 11 (64位)
- **内存:** 8 GB 或更高
- **硬盘:** SSD,至少 1 GB 可用空间
- **显示器:** 全高清 (1920x1080)

## 使用技术

- **Python 3.x** - 主要编程语言
- **CustomTkinter** - 现代用户界面框架
- **Supabase** - 数据库和身份验证服务
- **PyInstaller** - 应用程序打包工具
- **Inno Setup** - Windows 安装程序创建工具

## 安装

**不建议从源代码自行构建。** 请使用组织提供的官方安装程序。

## 安全性

系统遵循严格的数据安全原则:

- 数据库存储在 Supabase 上(通过 SOC 2 Type II 安全认证)
- 权限层次结构:管理员(Admin)、员工(Staff)、用户(User)
- 通过 HTTPS/SSL 协议连接加密
- 不存储明文密码
- 变更历史记录用于检查和监控
